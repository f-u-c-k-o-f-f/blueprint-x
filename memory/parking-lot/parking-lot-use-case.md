Parking Lot Example Use Cases

Appropriate holding area for valid observations, improvements, optimizations, alternative approaches, future features, and expansion opportunities that are not required to satisfy the current objective.

Example:

AI thinks: “Could support multiple users.
Could support teams.
Could support project history.
Could support cloud sync.
Could support agent collaboration.
Could support versioning.”

All of those might be good ideas.

None are required.

Therefore:

→ Parking Lot

Now imagine AI notices: “There is no way to export the build prompt.”

That’s different.

Without export:

Idea
 ↓
Discovery
 ↓
Build Prompt

...stuck.

The system cannot complete its purpose.

That is not a Parking Lot item.

That is a Critical Missing Requirement
